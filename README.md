## 📚 專案架構概覽



###  🎤 個人作品集 (Portfolio)
展示陳嘉暐老師在影視聲音設計、VR 實境音效及 AI 跨域實踐上的專業經歷與作品。
- **主要頁面**: `index.html` (中文), `en/index.html` (英文)
- **核心內容**:
    - **影視教育**: 課程資訊與教學理念。
    - **作品展示**: 收錄金馬獎、金穗獎獲獎之電影混音、紀錄片及 VR 聲音作品。
    - **AI 實踐**: 展示結合 Google Gemini, OpenAI 等 AI 技術的創新應用專案（如：靈曦籤苑、厭世占星卡牌）。
